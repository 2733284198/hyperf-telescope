### Get Started

## 一键安装telescope
`php ./bin/hyperf.php telescope:install`

## 如果一键安装有问题，请尝试下面安装方式：

### 复制migrations到根目录
`php ./bin/hyperf.php vendor:publish windawake/hyperf-telescope`

### 运行数据库表迁移
`php ./bin/hyperf.php migrate`

## 启动项目
`php ./bin/hyperf.php start`

## 访问地址
`http://127.0.0.1:9501/telescope/requests`
